# v0.2.4

MCP dry-run transformation planning for AI-assisted migration workflows.

## MCP transform plan tool

Expose dry-run transformation plans through MCP without mutating project files.

Audience: AI clients and consultants planning migration changes before execution

- Adds the MCP `emp.transformPlan` tool alongside `emp.analyze`, `emp.packs`, and `emp.benchmarks`.
- Runs existing EMP transformation planning in dry-run mode with no validation or file mutation.
- Returns pack, mode, status, selected engine, plan engine, summary, planned change count, changed files, recipes, replacement counts, and OpenRewrite command metadata.
- Adds an automated no-mutation test using a Spring fixture with a Jakarta namespace rewrite plan.
