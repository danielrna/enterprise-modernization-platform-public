## MCP benchmark evidence tool

Expose published benchmark evidence summaries through MCP.

Audience: AI clients and consultants comparing public modernization evidence

- Adds the MCP `emp.benchmarks` tool alongside `emp.analyze` and `emp.packs`.
- Summarizes published benchmark totals, checkout-backed evidence, catalog evidence, and validation pass/fail/skip/not-requested counts.
- Supports filtering by pack, source, validation status, and result limit.
- Returns concise per-report fields with slug, name, pack, readiness, source, validation status, confidence, finding count, report path, and public report URL.
