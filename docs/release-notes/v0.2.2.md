MCP pack metadata discovery for AI-assisted modernization planning.

## MCP pack metadata tool

Expose modernization pack discovery through MCP before running readiness analysis.

Audience: AI clients and consultants choosing the right modernization workflow

- Adds the MCP `emp.packs` tool alongside `emp.analyze`.
- Lists all available modernization packs with migration, vertical, category, version, and check-count metadata.
- Returns full pack metadata for one requested pack, including checks and report sections.
- Adds automated tests for tool discovery, pack listing, pack detail lookup, and unknown-pack errors.
